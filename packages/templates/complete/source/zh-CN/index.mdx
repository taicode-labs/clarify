# 使用 Clarify 构建多语言文档

这个页面演示了 Clarify 的原生 i18n 目录约定。默认语言内容位于 `{{contentDir}}/en-US`，中文内容位于 `{{contentDir}}/zh-CN`。

<Callout title="i18n 提示">
  当前模板只翻译了首页和写作指南，其他页面会根据 `missing: 'fallback'` 自动回退到英文内容。
</Callout>

## 目录结构

```txt
{{contentDir}}/
  en-US/
    index.mdx
    guides/writing-content.mdx
  zh-CN/
    index.mdx
    guides/writing-content.mdx
```
