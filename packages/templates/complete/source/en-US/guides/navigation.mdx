# Navigation

Clarify navigation is configured in `clarify.ts`. The starter uses explicit tabs so users can see how every part fits together.

## Tabs and sidebar groups

```ts title="clarify.ts"
export default defineConfig({
  tabs: [
    {
      tab: 'Docs',
      icon: 'BookOpen',
      pages: [
        {
          group: 'Start here',
          icon: 'Rocket',
          pages: [
            { page: 'index', title: 'Overview', icon: 'Sparkles' },
            { page: 'guides/writing-content', title: 'Writing content', icon: 'PenLine' },
          ],
        },
      ],
    },
  ],
})
```

## File tree mode

For smaller docs, use `FileTree` and let Clarify derive navigation from files.

```ts title="clarify.ts"
export default defineConfig({
  tabs: [
    {
      tab: 'Docs',
      pages: 'FileTree',
    },
  ],
})
```

## Navbar and footer

```ts title="clarify.ts"
export default defineConfig({
  navbar: {
    links: [
      { label: 'Guides', href: '/guides/writing-content' },
      { label: 'GitHub', href: 'https://github.com/taicode-labs/clarify', external: true },
    ],
  },
  footer: {
    copyright: '© 2026 Your Company.',
    links: [{ label: 'Configuration', href: '/reference/configuration' }],
    socials: { GitHub: 'https://github.com/taicode-labs/clarify' },
  },
})
```

## OpenAPI entries

Add an OpenAPI file to navigation with `openapi` instead of `page`.

```ts title="clarify.ts"
{
  openapi: 'api.openapi.json',
  title: 'OpenAPI reference',
  icon: 'FileJson2',
}
```

This renders the full reference for `{{contentDir}}/api.openapi.json` at `/api`.
