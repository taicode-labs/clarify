# MDX components

This page shows the built-in components available in MDX pages.

## Callout and note

<Callout title="Callout">

<Note>

## Properties

<Properties>
  <Property name="children" type="ReactNode">
    The content rendered inside the component.
  </Property>
  <Property name="title" type="string">
    A short label used by cards, callouts, or headings.
  </Property>
  <Property name="icon" type="string">
    Icon names are resolved from Lucide icons.
  </Property>
</Properties>
