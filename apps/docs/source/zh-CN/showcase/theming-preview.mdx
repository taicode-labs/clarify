# 主题定制预览

Clarify 支持通过配置文件选择主题预设，并在预设基础上覆盖品牌主色。本页只展示当前稳定的主题配置能力。

---

## 主题预设

```json
{
  "theme": {
    "preset": "default"
  }
}
```

`default` 使用 `#0ea5e9`（天蓝色）作为基础品牌主色。你也可以选择 `mint` 或 `violet`。

---

## 覆盖品牌主色

```json
{
  "theme": {
    "preset": "mint",
    "primary": "#14b8a6"
  }
}
```

`primary` 会在预设基础上覆盖品牌主色。

---

## 主题边界

当前版本不提供稳定的项目级自定义 CSS 入口，也不支持通过用户项目的 `tailwind.config.js` 覆盖 Clarify 渲染器样式。

如需更深层的品牌定制，应等待后续明确的主题或组件扩展 API。
