# Tabs / Tab

`Tabs` 在同一区域展示互斥内容，适合语言、包管理器、框架和平台差异。`defaultValue` 应匹配一个 `Tab` 的 `value`。

## 基础用法

```mdx
<Tabs defaultValue="pnpm">
  <Tab title="pnpm" value="pnpm">`pnpm add @clarify-labs/renderer`</Tab>
  <Tab title="npm" value="npm">`npm install @clarify-labs/renderer`</Tab>
</Tabs>
```

<Tabs defaultValue="pnpm">
  <Tab title="pnpm" value="pnpm">`pnpm add @clarify-labs/renderer`</Tab>
  <Tab title="npm" value="npm">`npm install @clarify-labs/renderer`</Tab>
</Tabs>

## 复杂内容

`Tab` 支持代码块、表格、列表和其他内置组件：

````mdx
<Tabs defaultValue="pnpm">
  <Tab title="pnpm" value="pnpm">

    ```bash
    pnpm add @clarify-labs/renderer
    ```

    查看 [快速开始](/getting-started) 继续配置。

  </Tab>
  <Tab title="npm" value="npm">
    <Collapse title="查看 npm 配置">这里支持 **Markdown**。</Collapse>
  </Tab>
</Tabs>
````

## 属性

### Tabs

<Properties>
  <Property name="defaultValue" type="string">首次渲染时选中的标签值。</Property>
  <Property name="children" type="ReactNode">一个或多个 `Tab`。</Property>
  <Property name="className" type="string">可选的外层容器类名。</Property>
</Properties>

### Tab

<Properties>
  <Property name="title" type="ReactNode">标签标题。</Property>
  <Property name="value" type="string">标签的稳定唯一值。</Property>
  <Property name="children" type="ReactNode">标签面板内容，支持完整 MDX。</Property>
</Properties>

<Callout type="tip">保持各标签内容结构相近。若每项内容很长或需要独立链接，应拆成不同页面。</Callout>
