# WebFrame

`WebFrame` 用于呈现产品页面、托管演示或静态 UI 预览，并让它与普通正文形成清晰区分。提供 `src` 时使用延迟加载的 iframe；不提供时直接渲染 `children`。

当 URL 或页面上下文有意义时保留浏览器外框；展示聚焦的组件预览时可以隐藏外框，减少无关视觉信息。

## 嵌入网页

```mdx
<WebFrame src="https://clarify.pub" title="Clarify 官网" height={360} />
```

<WebFrame src="https://clarify.pub" title="Clarify 官网" height={360} />

设置 `showWindow={false}` 可以隐藏模拟浏览器标题栏：

```mdx
<WebFrame src="https://clarify.pub" title="组件预览" showWindow={false} height={320} />
```

<WebFrame src="https://clarify.pub" title="组件预览" showWindow={false} height={320} />

## 外框样式

同一份预览既可以呈现为浏览器页面，也可以作为简洁的嵌入区域：

<WebFrame title="设置预览" height={150}>
  <div style={{ padding: 24 }}>
    <strong>工作区访问权限</strong>
    <p style={{ marginTop: 8 }}>控制哪些成员可以发布文档。</p>
  </div>
</WebFrame>

<WebFrame title="设置预览" showWindow={false} height={150}>
  <div style={{ padding: 24 }}>
    <strong>工作区访问权限</strong>
    <p style={{ marginTop: 8 }}>控制哪些成员可以发布文档。</p>
  </div>
</WebFrame>

## 内联预览

```mdx
<WebFrame title="内联预览" height={180}>
  <div style={{ padding: 24 }}>
    <strong>预览内容</strong>
    <p>内容直接渲染在框架内部。</p>
  </div>
</WebFrame>
```

带浏览器上下文的渲染效果：

<WebFrame title="内联预览" height={180}>
  <div style={{ padding: 24 }}>
    <strong>预览内容</strong>
    <p>内容直接渲染在框架内部。</p>
  </div>
</WebFrame>

## 属性

<Properties>
  <Property name="src" type="string">可选的 iframe URL，省略时渲染 `children`。</Property>
  <Property name="title" type="string">iframe 可访问名称和窗口标题。默认值为 `'Embedded content'`。</Property>
  <Property name="showWindow" type="boolean">是否显示模拟窗口外壳。默认值为 `true`。</Property>
  <Property name="height" type="CSSProperties['height']">内容高度。默认值为 `420`。</Property>
  <Property name="className" type="string">可选的外层容器类名。</Property>
  <Property name="iframeClassName" type="string">可选的 iframe 或内联容器类名。</Property>
  <Property name="children" type="ReactNode">未提供 `src` 时的可选内容。</Property>
</Properties>

其他原生 iframe 属性也会透传，例如 `allow`、`sandbox` 和 `referrerPolicy`。

<Callout type="warning">外部站点可能通过安全响应头禁止 iframe 嵌入。嵌入不受信任内容时应配置严格的 `sandbox`。</Callout>
