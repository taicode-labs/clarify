# Note

`Note` 是轻量的内联提示，适合不会改变任务流程的补充信息。它没有类型选择或标题栏，视觉权重低于 `Callout`。

## 用法

```mdx
<Note>
  标题、H2/H3 和导航命名会影响站内搜索结果。
</Note>
```

<Note>标题、H2/H3 和导航命名会影响站内搜索结果。</Note>

内容支持链接、行内代码和其他内联 Markdown。

## 属性

<Properties>
  <Property name="children" type="ReactNode">必填的提示内容。</Property>
</Properties>

<Callout type="tip">信息需要标题、语义颜色或风险等级时，改用 `Callout`。</Callout>
