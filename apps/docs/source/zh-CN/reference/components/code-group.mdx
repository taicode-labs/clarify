# CodeGroup

`CodeGroup` 把等价代码示例放在同一个代码面板中，适合不同包管理器、语言或运行环境。每个围栏代码块会成为一个可切换标签。

## 用法

````mdx
<CodeGroup title="安装 CLI">

```bash title="pnpm"
pnpm add -D @clarify-labs/cli
```

```bash title="npm"
npm install --save-dev @clarify-labs/cli
```

</CodeGroup>
````

<CodeGroup title="安装 CLI">

```bash title="pnpm"
pnpm add -D @clarify-labs/cli
```

```bash title="npm"
npm install --save-dev @clarify-labs/cli
```

</CodeGroup>

## 属性

<Properties>
	<Property name="title" type="string">可选的代码组标题。</Property>
	<Property name="children" type="ReactNode">必填。两个或更多围栏代码块。</Property>
</Properties>

代码块语言决定语法高亮，`title` 元数据决定标签文案。只有一个代码块时，直接使用普通代码围栏更简洁。

<Callout type="tip">各标签应完成同一任务。不同阶段的命令应使用 `Steps`，不要放进 `CodeGroup`。</Callout>
