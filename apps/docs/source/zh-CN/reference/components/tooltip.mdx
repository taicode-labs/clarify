# Tooltip

`Tooltip` 用于在原处解释简短术语，鼠标悬停或键盘聚焦时显示。

## 用法

```mdx
Clarify 在 Markdown 和 MDX 页面中支持 <Tooltip content="GitHub Flavored Markdown">GFM</Tooltip>。
```

Clarify 在 Markdown 和 MDX 页面中支持 <Tooltip content="GitHub Flavored Markdown">GFM</Tooltip>。

术语上方空间不足时，可以设置 `side="bottom"`。

## Props

<Properties>
	<Property name="content" type="ReactNode">必填的简短补充解释。</Property>
	<Property name="children" type="ReactNode">必填的可见且可通过键盘聚焦的触发内容。</Property>
	<Property name="side" type="'top' | 'bottom'">首选显示方向。默认值为 `'top'`。</Property>
	<Property name="className" type="string">可选的外层容器类名。</Property>
</Properties>

<Callout type="warning">不要把前置条件、警告、链接或长说明放进 Tooltip。触屏和辅助技术用户不应依赖发现隐藏内容才能理解页面。</Callout>
