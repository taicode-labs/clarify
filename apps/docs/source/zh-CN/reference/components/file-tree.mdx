# FileTree

`FileTree` 用于展示项目或目录结构，无需维护容易错位的制表字符。

## 用法

```mdx
<FileTree aria-label="项目文件">
  <FileTreeItem name="source" type="folder">
    <FileTreeItem name="index.mdx" description="首页" />
    <FileTreeItem name="guides" type="folder">
      <FileTreeItem name="deploy.mdx" />
    </FileTreeItem>
  </FileTreeItem>
</FileTree>
```

<FileTree aria-label="项目文件">
  <FileTreeItem name="source" type="folder">
    <FileTreeItem name="index.mdx" description="首页" />
    <FileTreeItem name="guides" type="folder"><FileTreeItem name="deploy.mdx" /></FileTreeItem>
  </FileTreeItem>
</FileTree>

## Props

### FileTree

<Properties>
  <Property name="aria-label" type="string">可选的无障碍标签。上下文没有说明目录内容时应提供。</Property>
  <Property name="children" type="ReactNode">一个或多个 `FileTreeItem` 元素。</Property>
</Properties>

`FileTree` 还支持原生 `div` 属性。

### FileTreeItem

<Properties>
  <Property name="name" type="ReactNode">必填的文件或目录名称。</Property>
  <Property name="type" type="'file' | 'folder'">项目图标，存在子项时自动推断为目录。</Property>
  <Property name="description" type="ReactNode">可选的简短用途或备注。</Property>
  <Property name="children" type="ReactNode">可选的嵌套文件和目录。</Property>
</Properties>
