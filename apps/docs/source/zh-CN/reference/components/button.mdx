# Button

`Button` 提供明确的下一步操作。传入 `href` 时渲染为链接，不传时渲染为原生按钮；站内链接会保留 Clarify 的客户端导航和当前语言。

## 用法

```mdx
<Button href="/getting-started" arrow="right">开始安装</Button>

<Button href="https://github.com/taicode-labs/clarify" variant="outline">
  查看 GitHub
</Button>
```

## 样式

`primary` 适合页面主行动，`secondary` 和 `outline` 适合次级行动，`filled` 使用主题主色，`text` 适合低视觉权重的行内入口。

## 属性

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `variant` | `'primary' \| 'secondary' \| 'filled' \| 'outline' \| 'text'` | `'primary'` | 视觉样式 |
| `arrow` | `'left' \| 'right'` | - | 显示方向箭头 |
| `href` | `string` | - | 存在时渲染为链接 |
| `children` | `ReactNode` | 必填 | 按钮文案 |
| `className` | `string` | - | 追加类名 |

组件还会透传原生 `<a>` 或 `<button>` 属性，例如 `target`、`rel`、`type` 和 `disabled`。

<Callout type="tip">一个区域通常只保留一个主按钮。仅用于跳转正文链接时，优先使用普通 Markdown 链接。</Callout>
