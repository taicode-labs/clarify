# 使用指南

Clarify 的用户指南按真实任务收敛为四篇核心文档，避免同一个能力分散在多个页面中。

---

## 推荐阅读顺序

1. [快速开始](/getting-started) — 安装 CLI、创建页面并构建静态站点。
2. [写作文档](/guides/writing-content) — 理解 MDX、Frontmatter、文件路由和内置组件。
3. [配置站点](/guides/navigation) — 配置导航、主题、多语言和部署子路径。
4. [API 文档](/features/openapi) — 用 OpenAPI 生成 Reference，并在指南中嵌入接口。
5. [发布上线](/guides/deployment) — 部署静态产物，并输出 `llms.txt` 给 AI 工具读取。

---

## 迁移场景

- [从 Mintlify 迁移](/guides/migrate-from-mintlify) — 将现有文档平滑迁移到 Clarify。
