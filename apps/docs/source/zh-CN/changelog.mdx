# 更新日志

---

## v0.8.0

### 新增

- 新增插件运行时 UI 插槽，支持 SSR 感知的懒加载，并可替换部分内置界面区域
- 新增 `build:assets` 插件 Hook，让插件能在生产构建中收集并输出静态资源
- 新增多语言 OpenAPI 虚拟模块，让多语言站点能渲染和嵌入对应语言的 API 规范
- 新增 Vitest 覆盖，覆盖 CLI 命令、配置加载、路由、SSG、OpenAPI、内容产物、SEO 产物、源码链接和 renderer 工具函数

### 改进

- 改进 OpenAPI 渲染，支持按页面注入 spec 数据、更清晰的虚拟模块生成和更稳定的 SSR 行为
- 改进 prose 表格渲染，增加表格包裹层、表头 nowrap 和表格单元格内 inline code 的布局保护
- 改进插件类型导出和运行时插槽命名，让集成边界更清晰
