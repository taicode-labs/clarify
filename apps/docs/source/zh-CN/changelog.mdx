# 更新日志

---

## v0.2.0

### 新增

- 新增 `Callout` 内置公开组件，用于信息、提示、警告、危险和成功状态说明
- 新增 `CardGroup` / `Card` 内置公开组件，用于首页入口、章节导航和功能矩阵
- 新增内置公开组件参考文档和组件演示页

### 改进

- `clarify init` 生成项目时默认引用 `@clarify-labs/cli@^0.2.0`

---

## v0.1.0

### 新增

- MDX 编译与渲染支持
- 基于文件系统的约定式路由
- 构建时静态生成（SSG）
- 客户端 Hydration 为 SPA
- 内置渲染器样式管线
- 配置文件系统（`clarify.ts` / `clarify.js` / `clarify.json`）
- 基础主题配置（`theme.primary`）
- 顶部导航配置
