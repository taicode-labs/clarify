# 主题定制预览

Clarify 支持通过 `clarify.json` 自定义主题。本页展示不同主题配置的效果。

---

## 默认主题

```json
{
  "theme": {
    "primary": "#0ea5e9"
  }
}
```

当前站点正在使用默认主题色 `#0ea5e9`（天蓝色）。

---

## 自定义 CSS

除了配置主题色，你还可以通过自定义 CSS 进一步调整样式：

```css
/* 自定义滚动条 */
.clarify-main::-webkit-scrollbar {
  width: 8px;
}

.clarify-main::-webkit-scrollbar-thumb {
  background: #cbd5e1;
  border-radius: 4px;
}
```
