# OpenAPI 文档渲染演示

本页展示 Clarify 渲染 OpenAPI 规范的效果。

---

## 自动生成的 API 页面

将 `api.openapi.json` 放入 `source/content/` 目录后，Clarify 会自动生成路由 `/api`，并使用 `OpenApiPage` 组件渲染完整的 API 文档。页面效果如下：

```
标题：Pages API
版本：1.0.0

GET    /pages        获取所有页面
POST   /pages        创建新页面
GET    /pages/{id}   获取单个页面
PUT    /pages/{id}   更新页面
DELETE /pages/{id}   删除页面
```

---

## 嵌入单个接口

在任意 MDX 页面中，使用 `OpenApiEndpoint` 组件嵌入特定接口。**无需导入，Clarify 的内置组件在 MDX 中可直接使用。**

```mdx
# 页面管理 API

<OpenApiEndpoint specPath="/api" path="/pages" method="get" />
```

渲染效果：

<OpenApiEndpoint specPath="/api" path="/pages" method="get" />
