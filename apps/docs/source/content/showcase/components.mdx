## MDX 组件

你可以在 Markdown 中直接使用 JSX：

<div className="bg-blue-50 border border-blue-200 p-4 rounded-lg">
  <h4 className="text-blue-800 font-semibold mb-2">提示</h4>
  <p className="text-blue-700">
    这是一个自定义 React 组件，直接在 MDX 中渲染！
  </p>
</div>
