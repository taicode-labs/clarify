# 环境准备与安装

从零开始搭建一个 Clarify 文档站点。

---

## 前置条件

- Node.js ≥ 20
- pnpm ≥ 9

---

## 安装

### 1. 创建项目

```bash
mkdir my-docs && cd my-docs
pnpm init
```

### 2. 安装依赖

```bash
pnpm add -D vite react react-dom react-router-dom @vitejs/plugin-react @tailwindcss/vite typescript
pnpm add -D @clarify/vite-plugin @clarify/renderer
```

### 3. 创建配置文件

**vite.config.ts**

```typescript
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import tailwindcss from '@tailwindcss/vite';
import { clarifyPlugin } from '@clarify/vite-plugin';

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
    clarifyPlugin(),
  ],
});
```

**clarify.json**（放在项目根目录）

```json
{
  "title": "我的文档",
  "description": "基于 Clarify 构建的文档站点"
}
```

### 4. 创建入口文件

**index.html**

```html
<!doctype html>
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8" />
    <title>我的文档</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/source/main.tsx"></script>
  </body>
</html>
```

**source/main.tsx**

```typescript
import { render } from '@clarify/renderer';
import { routes } from 'virtual:clarify-routes';
import { config } from 'virtual:clarify-config';

render({ config, routes });
```

### 5. 编写文档

在 `source/content` 目录下创建 `.mdx` 文件：

```bash
mkdir -p source/content
echo "# 首页\n\n欢迎来到我的文档！" > source/content/index.mdx
echo "# 指南\n\n这是指南页面。" > source/content/guide.mdx
```

### 6. 启动开发服务器

```bash
npx vite
```

打开浏览器访问 `http://localhost:5173`，你应该能看到自动生成的文档站点。
