# 主题定制预览

Clarify 支持通过配置文件设置品牌主色。本页只展示当前稳定的主题配置能力。

---

## 默认主题色

```json
{
  "theme": {
    "primary": "#0ea5e9"
  }
}
```

`#0ea5e9`（天蓝色）是当前站点配置中的品牌主色值。

---

## 其他主题色参考

| 颜色名称 | 色值 | 适合场景 |
|----------|------|----------|
| 翡翠绿 | `#10b981` | 自然、清新 |
| 紫罗兰 | `#8b5cf6` | 现代、科技 |
| 玫瑰红 | `#f43f5e` | 活力、醒目 |
| 琥珀橙 | `#f59e0b` | 温暖、友好 |
| 青柠绿 | `#84cc16` | 年轻、活力 |

---

## 主题边界

当前版本不提供稳定的项目级自定义 CSS 入口，也不支持通过用户项目的 `tailwind.config.js` 覆盖 Clarify 渲染器样式。

如需更深层的品牌定制，应等待后续明确的主题或组件扩展 API。
