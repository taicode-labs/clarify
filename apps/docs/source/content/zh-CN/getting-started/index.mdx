# 快速开始

欢迎来到 Clarify！本章将帮助你在几分钟内搭建并运行第一个文档站点。

---

## 推荐路径

1. [环境准备与安装](/getting-started/installation) — 准备 Node.js、pnpm 和 Clarify 依赖。
2. [创建你的第一个站点](/getting-started/first-site) — 从零创建一个可运行的文档站点。
3. [编写内容的最佳实践](/guides/writing-content) — 了解真实项目中如何组织内容。
4. [配置导航与侧边栏](/guides/navigation) — 把页面整理成适合读者的阅读路径。

## 一句话概括

Clarify = Vite + React + MDX + 约定式路由 + 静态生成。你只需写 Markdown，Clarify 负责剩下的所有事情。
