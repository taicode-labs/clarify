# 环境准备与安装

从零开始搭建一个 Clarify 文档站点。

---

## 前置条件

- Node.js ≥ 20
- pnpm ≥ 9

---

## 安装

### 1. 创建项目

```bash
mkdir my-docs && cd my-docs
pnpm init
```

### 2. 安装 CLI

```bash
pnpm add -D @clarify/cli
```

Clarify CLI 会封装开发服务器和静态构建流程，不需要手动安装或配置 Vite。

### 3. 初始化项目

```bash
pnpm exec clarify init
```

该命令会创建：

- `clarify.json`：站点展示配置
- `source/content/index.mdx`：首页内容
- `package.json` 中的 `dev` / `build` 脚本

你也可以手动创建 `clarify.json`：

```json
{
  "title": "我的文档",
  "description": "基于 Clarify 构建的文档站点"
}
```

### 4. 编写文档

在 `source/content` 目录下创建 `.md` 或 `.mdx` 文件：

```bash
mkdir -p source/content
echo "# 首页\n\n欢迎来到我的文档！" > source/content/index.mdx
echo "# 指南\n\n这是指南页面。" > source/content/guide.mdx
```

### 5. 启动开发服务器

```bash
pnpm dev
```

打开浏览器访问 CLI 输出的本地地址，你应该能看到自动生成的文档站点。
