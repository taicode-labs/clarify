# Developer Guide

This section is for contributors who want to understand or change Clarify itself.

---

## Repository layout

```txt
apps/
  docs/      Documentation playground and dev site
  www/       Marketing site
packages/
  cli/       CLI and docs engine
  renderer/  Shared React renderer and UI components
```

---

## Common commands

Use pnpm workspace commands from the repository root:

```bash
pnpm install
pnpm --filter @clarify-labs/docs build
```

Use `source/` for source files. Do not add new `src/` directories in this repository.
