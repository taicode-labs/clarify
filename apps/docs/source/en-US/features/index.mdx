# Feature Overview

Clarify is designed around one goal: publish high-quality technical documentation from content files that live with your project.

---

## Main capabilities

| Capability | What it does |
|------------|--------------|
| MDX content | Write Markdown and use React components inline. |
| File routes | Turn the `source/` directory into site routes. |
| OpenAPI | Generate full API Reference pages and embed single operations. |
| Static output | Build HTML, assets, copied public files, and `llms.txt`. |
| i18n | Organize localized content under locale folders such as `source/en-US`. |
| Configurable navigation | Define tabs, groups, pages, external links, and icons in `clarify.ts`. |
