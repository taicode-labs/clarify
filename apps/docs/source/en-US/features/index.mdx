# Feature Overview

Clarify puts content authoring, API reference generation, static publishing, and AI-readable documentation output into one local-first workflow.

---

## Core highlights

| Highlight | What Clarify provides | Continue reading |
|-----------|-----------------------|------------------|
| **Fast site setup** | `clarify dev`, `clarify build`, a default docs shell, file routes, and static output. | [Get Started](/getting-started) |
| **MDX authoring** | Frontmatter, React components, syntax highlighting, built-in Callout/Card/Code components, and directory-based routes. | [Writing Docs](/guides/writing-content) |
| **First-class OpenAPI** | `.openapi.json/.yaml/.yml` parsing, full API Reference pages, `OpenApiOperation` embedding, and build-time validation. | [API Documentation](/features/openapi) |
| **AI-ready publishing** | Raw Markdown/OpenAPI artifacts, page copy actions, raw-content links, and `llms.txt` discovery. | [Deployment](/guides/deployment) |

## Recommended path

1. **Start the site**: install the CLI, create `clarify.ts`, and add `source/`.
2. **Write content**: use MDX for home pages, guides, references, and component examples.
3. **Add APIs**: drop in OpenAPI specs, generate references, and embed key endpoints in guides.
4. **Shape information architecture**: configure tabs, groups, navigation, footer links, and locales.
5. **Publish for humans and AI**: build static HTML plus raw content files and `llms.txt`.
6. **Extend team workflows**: add plugins for governance, translation checks, search indexes, or custom artifacts.

---

## Design principles

1. **Task-oriented**: docs are organized around starting, writing, configuring, APIs, publishing, and extending.
2. **Content close to code**: MDX, OpenAPI, and config live in the repository for versioning and review.
3. **Static-first**: output can be hosted anywhere and consumed by search engines, knowledge bases, and AI tools.
4. **Progressive extension**: built-in behavior works by default; plugins are available when teams need more control.
