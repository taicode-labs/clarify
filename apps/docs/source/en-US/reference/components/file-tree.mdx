# FileTree

`FileTree` presents a project or directory structure without relying on fragile box-drawing text.

## Usage

```mdx
<FileTree aria-label="Project files">
  <FileTreeItem name="source" type="folder">
    <FileTreeItem name="index.mdx" description="Home page" />
    <FileTreeItem name="guides" type="folder">
      <FileTreeItem name="deploy.mdx" />
    </FileTreeItem>
  </FileTreeItem>
</FileTree>
```

<FileTree aria-label="Project files">
  <FileTreeItem name="source" type="folder">
    <FileTreeItem name="index.mdx" description="Home page" />
    <FileTreeItem name="guides" type="folder"><FileTreeItem name="deploy.mdx" /></FileTreeItem>
  </FileTreeItem>
</FileTree>

## Props

### FileTree

<Properties>
  <Property name="aria-label" type="string">Optional accessible label. Provide one when the surrounding text does not identify the tree.</Property>
  <Property name="children" type="ReactNode">One or more `FileTreeItem` elements.</Property>
</Properties>

`FileTree` also accepts native `div` props.

### FileTreeItem

<Properties>
  <Property name="name" type="ReactNode">Required file or directory name.</Property>
  <Property name="type" type="'file' | 'folder'">Item icon. Inferred as a folder when children are present.</Property>
  <Property name="description" type="ReactNode">Optional short inline purpose or note.</Property>
  <Property name="children" type="ReactNode">Optional nested files and directories.</Property>
</Properties>
