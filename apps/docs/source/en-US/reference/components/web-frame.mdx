# WebFrame

`WebFrame` presents a product page, hosted demo, or static UI preview without making it look like ordinary prose. It uses a lazy-loaded iframe when `src` is present and renders `children` directly otherwise.

Use the browser chrome when the URL or page context matters. Hide it for focused component previews where the surrounding browser metaphor would add noise.

## Embed a page

```mdx
<WebFrame src="https://clarify.pub" title="Clarify website" height={360} />
```

<WebFrame src="https://clarify.pub" title="Clarify website" height={360} />

Set `showWindow={false}` to hide the simulated browser header:

```mdx
<WebFrame src="https://clarify.pub" title="Component preview" showWindow={false} height={320} />
```

<WebFrame src="https://clarify.pub" title="Component preview" showWindow={false} height={320} />

## Frame styles

The same preview can be presented as a browser page or as a clean embedded surface:

<WebFrame title="Settings preview" height={150}>
  <div style={{ padding: 24 }}>
    <strong>Workspace access</strong>
    <p style={{ marginTop: 8 }}>Control who can publish documentation.</p>
  </div>
</WebFrame>

<WebFrame title="Settings preview" showWindow={false} height={150}>
  <div style={{ padding: 24 }}>
    <strong>Workspace access</strong>
    <p style={{ marginTop: 8 }}>Control who can publish documentation.</p>
  </div>
</WebFrame>

## Inline preview

```mdx
<WebFrame title="Inline preview" height={180}>
  <div style={{ padding: 24 }}>
    <strong>Preview content</strong>
    <p>This content renders directly inside the frame.</p>
  </div>
</WebFrame>
```

Rendered result with browser context:

## Props

<Properties>
  <Property name="src" type="string">Optional iframe URL; omit to render `children`.</Property>
  <Property name="title" type="string">Accessible iframe name and window title. Defaults to `'Embedded content'`.</Property>
  <Property name="showWindow" type="boolean">Whether to show the simulated window chrome. Defaults to `true`.</Property>
  <Property name="height" type="CSSProperties['height']">Content height. Defaults to `420`.</Property>
  <Property name="className" type="string">Optional class name for the outer container.</Property>
  <Property name="iframeClassName" type="string">Optional class name for the iframe or inline container.</Property>
  <Property name="children" type="ReactNode">Optional content when `src` is absent.</Property>
</Properties>

Other native iframe props are forwarded, including `allow`, `sandbox`, and `referrerPolicy`.

<Callout type="warning">External sites can block iframe embedding with security headers. Apply a restrictive `sandbox` when embedding untrusted content.</Callout>
