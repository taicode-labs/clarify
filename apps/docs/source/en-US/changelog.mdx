# Changelog

This page records notable changes to Clarify.

---

## 0.1.x

- Introduced the Clarify documentation app.
- Added MDX content rendering and file-based routing.
- Added OpenAPI Reference generation from `.openapi.json`, `.openapi.yaml`, and `.openapi.yml` files.
- Added localized documentation structure with `zh-CN` and `en-US` support.
- Added static build output with copied assets and `llms.txt`.
