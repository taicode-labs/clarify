# Changelog

---

## v0.2.0

### Added

- Added the `Callout` built-in public component for info, tip, warning, danger, and success notes
- Added the `CardGroup` / `Card` built-in public components for homepage entries, section navigation, and feature matrices
- Added built-in public component reference docs and a component demo page

### Improved

- `clarify init` now references `@clarify-labs/cli@^0.2.0` by default when generating projects

---

## v0.1.0

### Added

- MDX compilation and rendering support
- Convention-based, file-system routing
- Static generation at build time (SSG)
- Client-side hydration into an SPA
- Built-in renderer styling pipeline
- Configuration file system (`clarify.ts` / `clarify.js` / `clarify.json`)
- Basic theme configuration (`theme.preset` and `theme.tokens.colors.primary`)
- Top navigation configuration
