## Built-in public components

Clarify provides a set of components that can be used directly in MDX without manual imports. For the complete API, see [Built-in public components](/reference/built-in-components).

### Callout

`Callout` is used to emphasize explanations, risk warnings, recommended practices, or operation results. Common types include `info`, `note`, `tip`, `warning`, `danger`, and `success`.

```mdx
<Callout type="info">
  The model list is updated continuously. Before production integration, use the models currently available in the console or model marketplace as the source of truth.
</Callout>
```

<Callout type="info">
  The model list is updated continuously. Before production integration, use the models currently available in the console or model marketplace as the source of truth.
</Callout>

<Callout type="warning" title="Note">
  Before publishing to production, confirm that secrets, domains, permissions, and caching policies have all been validated.
</Callout>

<Callout type="success" title="Completed">
  After the build passes, Clarify generates directly deployable static HTML for every route.
</Callout>

### CardGroup / Card

`CardGroup` and `Card` are used to create section entries, feature matrices, or homepage navigation. The `icon` prop of `Card` uses lucide icon names and supports kebab-case syntax such as `badge-info`.

```mdx
<CardGroup cols={2}>
  <Card title="Quick start" icon="rocket" href="/getting-started">
    Install the CLI and create your first Clarify documentation site.
  </Card>
  <Card title="Writing docs" icon="file-code-2" href="/guides/writing-content">
    Organize content with Markdown, JSX, and built-in components.
  </Card>
</CardGroup>
```

## Custom MDX components

You can also use JSX directly in Markdown or import your own React components:

<div className="bg-blue-50 border border-blue-200 p-4 rounded-lg dark:border-blue-400/30 dark:bg-blue-400/10">
  <h4 className="text-blue-800 font-semibold mb-2 dark:text-blue-200">Tip</h4>
  <p className="text-blue-700 dark:text-blue-100">
    This is a custom React component rendered directly in MDX!
  </p>
</div>
